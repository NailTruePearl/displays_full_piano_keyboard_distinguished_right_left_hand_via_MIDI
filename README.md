This is a beta version program that guides you through playing piano pieces.<br>
After reading a MIDI file, it displays the piano keys you need to play.<br>
Keys for the left hand are shown in green, and keys for the right hand are shown in red.<br>
When the program runs, it displays the full keyboard.<br>
Many people find it very difficult to learn pieces that are not in C major;<br>
this program directly shows you which keys to press, making it very intuitive.<br>
You can select the piece you want to learn and adjust the playback speed.<br>
The program also labels every black key with its symbol for easy reference.<br>
The program does not require any external images—the entire interface is generated purely by code.<br>
The staff notation displayed on the program's interface is intended only for beginners learning pieces confirmed to be in C major;<br>
if you are unsure whether the piece is in C major, please ignore the staff display.<br>
